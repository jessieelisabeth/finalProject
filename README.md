# PROSPECTS
Prospects is a satirical dating simulator that asks the player to complete a bundle while managing a bustling personal life.
## Objective
Complete the bundle of joy before the end of day 7.
The required bundle slots are:

- Friendship Bracelet
- Letter of Acceptance
- Rose or Sunflower
- Thank You Note
- Cafe Gift Card

The sunflower is a hidden slot that represents personal growth.

## Play
The player can partake in several actions. Each of these actions take energy.
- View player stats
- View inventory
- View bundle progress
- Travel between five locations
- Talk to love interests
- Talk to friends
- Donate items
- End the current day
- Use unHinge as a shortcut
- Unlock game tips and the Bundle Planner after Day 5

Ending the day restores energy.

## Locations

- Party
- Cafe
- Classroom
- Gym
- Bedroom

## Unhinge
Unhinge is the Joja equivalent to this project's theme.
